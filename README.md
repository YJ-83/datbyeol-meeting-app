닻별 모임의 회원 정보 및 회비를 체계적으로 관리하는 웹 애플리케이션입니다.
**별도 설치 없이 브라우저에서 바로 실행**할 수 있으며, 모든 데이터는 **사용자의 브라우저에만 저장**됩니다.

## ✨ 주요 기능

### 1. 👥 회원 관리
- 회원 추가/수정/삭제
- 성명, 주민번호, 핸드폰번호, 가입일 관리
- 주민번호 마스킹 처리 (뒷자리 *로 표시)
- 자동 포맷팅 (주민번호: 000000-0000000, 핸드폰: 010-0000-0000)
- **📊 엑셀(XLSX) 파일로 내보내기** - 회원명부를 Excel 파일로 저장

### 2. 💰 회비 관리
- **모임별 회비 관리** - 여러 모임을 만들고 각각 회비 내역 관리
  - 예: "2026년 2월 정기모임", "봄 야유회", "송년회" 등
- 모임 추가/삭제 기능
- 회비 납부 내역 추가/수정/삭제
- 회원별 납부 이력 관리
- 납부일, 납부액, 납부방법, 비고 기록
- 모임별 총 납부액 및 건수 자동 집계

### 3. 📊 통계
- 총 회원 수
- 전체 모임의 총 회비 납부액
- 평균 회비
- 최근 납부일

### 4. 💾 데이터 관리
- **Excel(XLSX) 내보내기** - 회원명부를 엑셀 파일로 저장
- JSON 파일로 내보내기 (전체 데이터 백업)
- JSON 파일에서 가져오기 (데이터 복원/병합)
- 로컬 스토리지에 자동 저장 (브라우저 내 데이터 보존)

## 사용 방법

### 설치 및 실행
1. 파일 다운로드
   ```
   meeting_app/
   ├── index.html
   ├── style.css
   ├── app.js
   └── backups/ (자동 생성)
   ```

2. 브라우저에서 `index.html` 파일 열기
   - 더블클릭하여 실행
   - 또는 브라우저로 드래그 앤 드롭

### 회원 추가
1. "회원 관리" 탭 선택
2. "+ 회원 추가" 버튼 클릭
3. 정보 입력 (성명, 주민번호, 핸드폰번호, 가입일)
4. "저장" 버튼 클릭

### 회원명부 엑셀 내보내기
1. "회원 관리" 탭 선택
2. "📊 엑셀 내보내기" 버튼 클릭
3. Excel 파일 자동 다운로드 (예: `닻별_회원명부_2026-02-16.xlsx`)

### 모임 만들기
1. "회비 관리" 탭 선택
2. "+ 새 모임 만들기" 버튼 클릭
3. 모임 이름 및 날짜 입력 (예: "2026년 2월 정기모임")
4. "생성" 버튼 클릭

### 회비 추가
1. "회비 관리" 탭에서 모임 선택
2. "+ 회비 추가" 버튼 클릭
3. 정보 입력 (회원명 선택, 납부일, 납부액, 납부방법, 비고)
4. "저장" 버튼 클릭

### 모임 전환
- "회비 관리" 탭에서 드롭다운 메뉴로 모임 선택
- 각 모임별로 독립적인 회비 내역 유지

### 데이터 백업
1. "통계" 탭 선택
2. "JSON 내보내기" 버튼 클릭
3. 파일 저장 (예: `닻별_모임데이터_2026-02-16.json`)

### 데이터 복원
1. "통계" 탭 선택
2. "JSON 가져오기" 버튼 클릭
3. 백업 파일 선택
4. 덮어쓰기 또는 추가 선택

## 기술 스택
- **HTML5**: 구조
- **CSS3**: 디자인 (그라디언트, 애니메이션)
- **JavaScript (ES6+)**: 로직
- **SheetJS (XLSX)**: Excel 파일 생성
- **LocalStorage**: 데이터 저장

## 보안 고려사항
- **주민번호 마스킹**: 화면에 뒷자리 7자리는 '*'로 표시
- **Excel 파일**: 주민번호 전체가 포함되므로 파일 관리 주의
- **로컬 저장**: 데이터는 사용자의 브라우저에만 저장 (외부 전송 없음)
- **주의**: 주민번호는 민감 정보이므로, 공용 컴퓨터에서 사용 시 주의 필요

## 브라우저 호환성
- Chrome, Edge, Safari, Firefox 최신 버전 지원
- 모바일 브라우저 지원 (반응형 디자인)
- SheetJS 라이브러리 CDN 사용 (인터넷 연결 필요)

## 백업 파일
모든 소스 코드는 `backups/` 폴더에 자동 백업됩니다.
- 형식: `파일명.YYYYMMDD_HHMMSS.bak`
- 예: `app.js.20260216_143022.bak`
